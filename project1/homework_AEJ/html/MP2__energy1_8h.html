Retrieves a specific Two-Electron Integral (ERI) from sparse arrays.

This function searches for the integral \( (pq|rs) \) within the provided sparse lists (index and value).

Algorithm details: Since integrals are often stored exploiting 8-fold permutational symmetry (e.g., \( (pq|rs) = (rs|pq) = (qp|rs) \dots \)), this function:

  1. Canonicalizes the input indices p, q, r, s.
  2. Iterates through the index array to find the match.
  3. Returns the corresponding value if found, or 0.0 otherwise.

Retrieves a specific Two-Electron Integral (ERI) from sparse arrays.

This function handles the 8-fold permutational symmetry of real ERIs:

\[ (pq|rs) = (qp|rs) = (pq|sr) = (qp|sr) = (rs|pq) = (sr|pq) = (rs|qp) = (sr|qp) \]

The algorithm works as follows:

Step 1: Define Permutations Since we don't know which canonical version is stored in the file, we prepare all 8 possible combinations of the requested indices (p,q,r,s).

int permutations[8][4] = {
{p, q, r, s}, {q, p, r, s},
{p, q, s, r}, {q, p, s, r},
{r, s, p, q}, {s, r, p, q},
{r, s, q, p}, {s, r, q, p}
};

Step 2: Linear Search We iterate through the sparse list of integrals. For each stored integral, we check if its indices match ANY of our 8 permutations.

// ... check if stored indices match any permutation ...
